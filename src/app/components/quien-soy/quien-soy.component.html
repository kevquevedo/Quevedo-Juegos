Juego Propio

Generala Obligada

La Generala Obligada es un juego de azar. Se juega con 5 dados.
Cada jugador realiza un tiro para cada jugada, las cuales se explicaran mas adelante, y se determina que puntaje obtiene sobre cada una de ellas.

Las jugadas con las siguientes:
Jugada de 1

Por cada 1 que se saque se suma el valor del dado, en este caso, 1 punto por cada dado.

Jugada de 2

Por cada 2 que se saque se suma el valor del dado, en este caso, 2 puntos por cada dado.

Jugada de 3

Por cada 3 que se saque se suma el valor del dado, en este caso, 3 puntos por cada dado.

Jugada de 4

Por cada 4 que se saque se suma el valor del dado, en este caso, 4 puntos por cada dado.

Jugada de 5

Por cada 5 que se saque se suma el valor del dado, en este caso, 5 puntos por cada dado.

Jugada de 6

Por cada 6 que se saque se suma el valor del dado, en este caso, 6 puntos por cada dado.

Escalera

Si se sacan los 5 dados de menor a mayor, el jugador obtiene 20 puntos.

Full

El jugador que saque los 5 dados de menor a mayor, obtiene 20 puntos.

Poker

Si se sacan 4 dados iguales, el jugador obtiene 40 puntos.

Generala

Si se sacan 5 dados iguales, el jugador obtiene 50 puntos.

Generala Doble

Si vuelve a sacar 5 dados iguales, el jugador obtiene 100 puntos.
Si para esta ocasión, el jugador no obtuvo la generala en la tirada anterior, por mas que saque generala en esta tirada no suma puntos.
